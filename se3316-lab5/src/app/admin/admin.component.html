DMCA Takedown Procedure Tools Guide 
        
            
           This list describes the tools you can use for a DMCA Takedown Process
            
        
    
 Hiding Content: You will be able to hide content that has been striked with AUP strikes.  
 Restoring Content: You will be able to re-display hidden content that was claimed as copyright
            Documenting notices: As Administrator, you can use the "notices" tool to manage all disputes and notices received or claimed, and all the relevant information regarding them.
